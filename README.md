# elementor-pro
Mirror of the Elementor Pro WordPress plugin. You need a valid pro license to use this plugin.

## Composer.json setup:
```
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/frank-holub/elementor-pro"
    }
```

## Composer require:
```
composer require frank-holub/elementor-pro
```
